1) Δομές ελέγχου

Οι δομές ελέγχου (control structures) χρησιμοποιούνται για τον έλεγχο της ροής του προγράμματος. Αυτές οι δομές επιτρέπουν την εκτέλεση συγκεκριμένων εντολών βάσει κάποιων συνθηκών.

Οι λογικοί τελεστές (&&, ||, !) χρησιμοποιούνται για να συνδυάσουν συνθήκες και να ελέγξουν τη λογική αλήθειας ή ψεύδους στις εκφράσεις των δομών if-else.

Στη C++, οι βασικές δομές ελέγχου είναι οι εξής:

Δομή if: Η δομή if χρησιμοποιείται για τον έλεγχο μιας συνθήκης και την εκτέλεση κώδικα αν η συνθήκη είναι αληθής. 

                        
                                if (condition) {
                                // Κώδικας που εκτελείται αν η συνθήκη είναι αληθής
                                }

Δομή if-else: Η δομή if-else επιτρέπει την εκτέλεση διαφορετικού κώδικα ανάλογα με το αν η συνθήκη είναι αληθής ή ψευδής. (αν...αλλιώς) 

                            
                                if (condition) {
                                    // Κώδικας που εκτελείται αν η συνθήκη είναι αληθής
                                } else {
                                    // Κώδικας που εκτελείται αν η συνθήκη είναι ψευδής
                                }

Δομή if-else if-else...else: Χρησιμοποιείται για τον έλεγχο πολλών συνθηκών με διαδοχικά if-else if-else και στο τέλος ένα τελικό else. 

                             //μπορούμε να έχουμε από 3 και πάνω συνθήκες
                                if (condition1) {
                                    // Κώδικας που εκτελείται αν η condition1 είναι αληθής
                                } else if (condition2) {
                                    // Κώδικας που εκτελείται αν η condition2 είναι αληθής
                                } else {
                                    // Κώδικας που εκτελείται αν καμία από τις προηγούμενες συνθήκες δεν ικανοποιείται
                                }

Δομή switch: Η δομή switch χρησιμοποιείται για να εκτελέσει κώδικα βάσει της τιμής μιας έκφρασης. 

                
                    switch (expression) {
                        case value1:
                            // Κώδικας που εκτελείται αν η το 'expression' ισούται με value1
                            break;
                        case value2:
                            // Κώδικας που εκτελείται αν το 'expression' ισούται με value2
                            break;
                        //μπορούμε να βάλουμε και άλλα cases ...
                        default:
                            // Κώδικας που εκτελείται αν καμία από τις προηγούμενες περιπτώσεις δεν ικανοποιείται
                    }

Παρακάτω παραθέτουμε ένα παράδειγμα κώδικα που χρησιμοποιεί δομές ελέγχου (if , if-else και switch): 

                
                    #include <iostream>

                        int main() {
                            // Δομή if
                            int number = 10;
                            if (number > 5) {
                                std::cout << "Ο αριθμός είναι μεγαλύτερος από 5." << std::endl;
                            }
                        
                            // Δομή if-else
                            int x = 5;
                            if (x % 2 == 0) {
                                std::cout << "Ο αριθμός είναι άρτιος." << std::endl;
                            } else {
                                std::cout << "Ο αριθμός είναι περιττός." << std::endl;
                            }
                        
                            // Δομή switch
                            int choice = 2;
                            switch (choice) {
                                case 1:
                                    std::cout << "Επιλέξατε την περίπτωση 1." << std::endl;
                                    break;
                                case 2:
                                    std::cout << "Επιλέξατε την περίπτωση 2." << std::endl;
                                    break;
                                default:
                                    std::cout << "Επιλογή εκτός εύρους." << std::endl;
                            }
                        
                            return 0;
                        }

2) Δομές επανάληψης

Ο λόγος χρήσης των δομών αυτών είναι για να μην επαναλαμβάνουμε πολλαπλές φορές τις ίδιες εντολές ειδικά όταν αυτό μπορεί να γίνει εκατοντάδες ή και χιλιάδες φορές οπότε από ένα σημείο και μετά δεν είναι πρακτική η χρήση πολλών ίδιων εντολών αλλά μιας μόνο επανάληψης.

Υπάρχουν τρεις βασικές δομές επανάληψης στην C++:

Δομή επανάληψης for : Η δομή for χρησιμοποιείται για να επαναλάβετε μια σειρά εντολών για έναν ορισμένο αριθμό φορών με χρήση ενός index. Η σύνταξη της είναι: 

                    
                        for(αρχικοποίηση; συνθήκη; βήμα) {
                            // Κώδικας που επαναλαμβάνεται
                        }
                        Παράδειγμα:
                        
                        
                        for (int i = 0; i < 5; i++) {
                            cout << "Επανάληψη #" << i << endl;
                        }

Στο πάνω παράδειγμα for o δείκτης επανάληψης i αρχικοποιείτε στο 0 αλλά και δηλώνεται πρώτη φορά μέσα στην for. Αυτό είναι κάτι που μπορεί να γίνει στην c++ αλλά αν θέλουμε μπορούμε να δηλώσουμε την μεταβλητή δείκτη σε προηγούμενο μέρος του κώδικα και απλά να αρχικοποιηθεί μέσα. Δηλάδή: 

                        
                            int i;
                            for (i = 0; i < 5; i++) {
                                cout << "Επανάληψη #" << i << endl;
                            }

Δομή επανάληψης while: : Η δομή while χρησιμοποιείται για να επαναλάβετε μια σειρά εντολών εφόσον μια συγκεκριμένη συνθήκη είναι αληθής. Η σύνταξη της είναι: 

                        
                            while (συνθήκη) {
                                // Κώδικας που επαναλαμβάνεται
                            }

Η while έχει την ιδιότητα να μπορεί να γίνει ισοδύναμη με την for , αρκεί να δηλώσουμε έναν δείκτη/μεταβλητή πριν την while, να την αρχικοποιήσουμε , η συνθήκη να είναι αντίστοιχη με αυτή της for και τέλος να υπάρχει εντολή μέσα της για το βήμα επανάληψης. Για παράδειγμα, έχουμε τον προηγούμενο κώδικα με while αντί για for: 

                        
                            int i=0;
                            while(i < 5) {
                                cout << "Επανάληψη #" << i << endl;
                                i++; // βήμα επανάληψης : i = i+1;
                            }

Δομή επανάληψης do...while: : Η δομή do...while είναι παρόμοια με τη δομή while, αλλά ελέγχει τη συνθήκη μετά από κάθε εκτέλεση του κώδικα. Αυτό σημαίνει ότι ο κώδικας εκτελείται τουλάχιστον μία φορά, ακόμη κι αν η συνθήκη δεν είναι αληθής. Η σύνταξη της είναι: 

                            
                                do {
                                    // Κώδικας που επαναλαμβάνεται
                                } while (συνθήκη);

Παράδειγμα: 

                            
                            int i=0;
                             do {
                                cout << "Επανάληψη #" << i << endl;
                                i++; // βήμα επανάληψης : i = i+1;
                            }while(i < 5);

3) Πίνακες

Πίνακες (Arrays):

Ορισμός πίνακα:Ο πίνακας είναι μια συλλογή στοιχείων του ίδιου τύπου που αποθηκεύονται σε συνεχόμενες θέσεις μνήμης. Για παράδειγμα, μπορείτε να ορίσετε έναν πίνακα ακεραίων: int numbers[5]; που περιέχει 5 ακέραιους.

Πρόσβαση στα στοιχεία του πίνακα: Τα στοιχεία του πίνακα προσπελαύνονται με τη χρήση αγκύλων '[]'μετά από το όνομα του πίνακα. Για παράδειγμα, numbers[0] θα προσπελάσει το πρώτο στοιχείο του πίνακα , όπου το 0 εδώ είναι το index του πίνακα (δείκτης) το οποίο σημαίνει πως τώρα δείχνει στο πρώτο στοιχείο του (στην c++ οι πίνακες ξεκινούν από το index 0 μέχρι το μεγεθος-1).

Στη συνέχεια που θα μιλήσουμε για του δείκτες θα δούμε πως στην πραγματικότητα οι πίνακες είναι δείκτες και ότι υπάρχει και άλλος τρόπος προσπέλασης και χρήσης γενικά των πινάκων.

Δήλωση πίνακα με σταθερό μέγεθος (στατικός πίνακας): Σε πολλές περιπτώσεις, ο πίνακας δηλώνεται με έναν σταθερό αριθμό στοιχείων που δεν μπορεί να αλλάξει κατά τη διάρκεια της εκτέλεσης του προγράμματος (όπως και στα πάνω παραδείγματα).

Δυναμική δέσμευση μνήμης για πίνακες (δυναμικός πίνακας): Σε άλλες περιπτώσεις, ο πίνακας μπορεί να δημιουργηθεί δυναμικά κατά την εκτέλεση του προγράμματος χρησιμοποιώντας τον τελεστή new ή την συνάρτηση βιβλιοθήκης (από τη c) malloc.

Αρχικοποίηση πίνακα: Μπορείτε να αρχικοποιήσετε τα στοιχεία του πίνακα κατά τη δήλωσή του ή αργότερα, μετά τη δέσμευση της μνήμης. Δήλωση πίνακα ακεραίων με 5 θέλεις μνήμης(στατικά): int numbers[5]; Αργότερα μπορούμε να δώσουμε τιμές στον πίνακα , συνήθως με τη χρήση δομών επανάληψης.

Ακολουθούν δύο παραδείγματα χρήσης πίνακα σε κώδικα.

Παράδειγμα 1: Υπολογισμός του μέσου όρου των αριθμών σε έναν πίνακα. 

                        
                            int main() {
                                int numbers[] = {10, 20, 30, 40, 50}; // εκχώρηση όλων των στοιχείων σε πίνακα κατά τη δήλωση
                                int sum = 0;
                                const int size = 5; // όσο και το μέγεθος του πίνακα
                            
                                for (int i = 0; i < size; i++) {
                                    sum += numbers[i];
                                }
                            
                                double average = (double)sum/ size; // κανονική διαίρεση και όχι διαίρεση div , που απλά θα έδινε το πυλίκο χωρίς να συνεχίσει
                                //αφαιρώντας έτσι το δεκαδικό μέρος από το αποτέλεσμα
                                //αυτό επιτυγχάνεται με το casting του ακεραίου sum σε double μόνο για αυτή τη διαίρεση
                            
                                std::cout << "Ο μέσος όρος των αριθμών είναι: " << average << std::endl;
                            
                                return 0;
                            }

Παράδειγμα 2: Αναζήτηση αριθμού σε πίνακα. 

                        
                            #include <iostream>

                                int main() {
                                    int numbers[] = {10, 20, 30, 40, 50};
                                    int target = 30;
                                    const int size = 5;
                                    bool found = false;
                                
                                    for (int i = 0; i < size; i++) {
                                        if (numbers[i] == target) {
                                            found = true;
                                            break;
                                        }
                                    }
                                
                                    if (found) {
                                        std::cout << "Ο αριθμός " << target << " βρέθηκε στον πίνακα." <<  std::endl;
                                    } else {
                                        std::cout << "Ο αριθμός " << target <<" δεν βρέθηκε στον πίνακα." << std::endl;
                                    }
                                
                                    return 0;
                                }

Αυτό το παράδειγμα αναζητά έναν συγκεκριμένο αριθμό , target = 30, σε έναν πίνακα πέντε στοιχείων και εμφανίζει αν βρέθηκε ή όχι. Αν βρεθεί μέσα στην επανάληψη που γίνονται έλεγχος αν κάθε στοιχείο του πίνακα είναι ίσο με το target τότε η bool μεταβλητή found από false σε true τερματίζοντας επίσης την επανάληψη for με την εντολή break. Ανάλογα αν παραμείνει false τότε το if στο τέλος δεν ισχύει οπότε εκτελείτε μόνο το else μέρος της δομλης ελέγχου if-else. Αν από την άλλη βρεθεί κάπου τότε αλλάζει τότε η found θα γίνει true και θα εκτελεστεί το if μέρος.

4) Συμβολοσειρές

Στη γλώσσα προγραμματισμού C++ (αντίστοιχα όπως και στη c), οι συμβολοσειρές (ή strings) αναπαριστώνται ως ακολουθίες χαρακτήρων που τερματίζονται με έναν ειδικό χαρακτήρα, τον χαρακτήρα '\0' (NULL). Αυτός ο χαρακτήρας υποδηλώνει το τέλος της συμβολοσειράς.

Μπορούμε να δηλώσουμε μια συμβολοσειρά ως εξής: 

 char myString[] = "Αυτή είναι μια συμβολοσειρά";

Εδώ, η myString είναι μια συμβολοσειρά που περιέχει την πρόταση "Αυτή είναι μια συμβολοσειρά".

Για την εκτύπωση της συμβολοσειράς στην οθόνη, μπορoύμε να χρησιμοποιήσουμε την εντολή cout της βιβλιοθήκης iostream όπως στο παρακάτω παράδειγμα. 

                    
                        #include <iostream>

                            int main() {
                                char myString[] = "This is a string";
                                std::cout << myString << std::endl;
                                return 0;
                            }

Τέλος, μπορούμε να εκτελέσουμε διάφορες βασικές λειτουργίες σε συμβολοσειρές, όπως την συνέννωση δύπ συμβολοσειρών τον υπολογιστών τους μήκους τους και πολλά άλλα. Αυτές οι λειτουργίες υπάρχουν στη βιβλιοθήκη cstring (από την c) ως συναρτήσεις.

Ένα παράδειγμα χρήσης κάποιων συναρτήσεων της cstring: 

                    
                        #include <iostream>
                        #include <cstring> // Για την χρήση συναρτήσεων σχετικές με τις συμβολοσειρές
                            
                            int main() {
                                char myString1[] = "This is a ";
                                char myString2[] = "string";
                            
                                // Συνένωση δύο συμβολοσειρών
                                strcat(myString1, myString2);
                            
                                std::cout << myString1  << std::endl; // στην πρώτη συμβολοσειρά γίνεται η συνένωση 
                                //άρα τώρα ισχύει:myString1 = "This is a string" , με '\0' = null character στο τέλος
                            
                                // Εύρεση του μήκους της συμβολοσειράς
                                int length = strlen(myString1);
                                std::cout << "Το μήκος της συμβολοσειράς είναι: " << length << std::endl;
                                //θα πρέπει να έχει το συνολικό μήκος των δύο πάνω συμβολοσειρών
                                //θα εμφανίζει 14 ως το μήκος
                            
                                return 0;
                            }

Στον παραπάνω κώδικα, χρησιμοποιήσαμε τις συναρτήσεις strcat και strlen από τη βιβλιοθήκη cstring για τη συνένωση δύο συμβολοσειρών και τον υπολογισμό του μήκους της τελικής αντίστοιχα.

Παράρτημα: Βασικές Συναρτήσεις της βιβλιοθήκης cstring

Συνάρτηση Περιγραφή
char *strcpy(char *destination, const char *source) Αντιγράφει μια συμβολοσειρά σε μια άλλη.
char *strcat(char *destination, const char *source) Συνενώνει δύο συμβολοσειρές.
size_t strlen(const char *str) Επιστρέφει το μήκος μιας συμβολοσειράς.
int strcmp(const char *str1, const char *str2) Συγκρίνει δύο συμβολοσειρές.
char *strchr(const char *str, int c) Επιστρέφει ένα δείκτη στο πρώτο στοιχείο της συμβολοσειράς όπου εμφανίζεται ο συγκεκριμένος χαρακτήρας (ή τον τερματικό χαρακτήρα '\0' εάν δεν βρεί τον χαρακτήρα).
