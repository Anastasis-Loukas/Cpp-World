Κεφάλαιο 1ο - Εισαγωγή

1) Σύντομη ιστορική αναδρομή

Η C++ είναι μία αντικειμενοστραφής γλώσσα προγραμματισμού η οποία δημιουργήθηκε το 1979 από τον Bjarne Stroustrup. Το αρχικό της όνομα ήταν "C with classes" καθώς αποτελεί υπερσύνολο της C ενώ 5 χρόνια αργότερα μετονομάστηκε σε "C++". Κύριο χαρακτηριστικό της γλώσσας είναι η οργάνωση των δεδομένων και των λειτουργιών σε κλάσεις. Τα αντικείμενα είναι στιγμιότυπα των κλάσεων αυτών και αποτελούν το κύριο δομικό στοιχείο της γλώσσας. Επιπρόσθετα χαρακτηριστικά της C++ είναι τα εξής:

  • Ενθυλάκωση: Οργάνωση των δεδομένων και των συναρτήσεων της κλάσης, ώστε να επιτυγχάνεται απόκρυψη πληροφορίας
  • Πολυμορφισμός: Η δυνατότητα της γλώσσας να χρησιμοποιεί το ίδιο συμβολικό όνομα για διαφορετικές χρήσεις
  • Κληρονομικότητα: Η δυνατότητα των κλάσεων να κληρονομούνται από άλλες κλάσεις και να προσθέτουν την δική τους λειτουργικότητα

2) Το πρώτο μας πρόγραμμα

// Το πρώτο μας πρόγραμμμα

#include <iostream>

using namespace std;

int main()
{
  cout<<"Hello C++ World\n";
  return 0;
}

Επεξήγηση εντολών

Γραμμή 1η: Η πρώτη γραμμή αποτελεί ένα σχόλιο. Τα σχόλια μίας γραμμής ξεκινάνε με τους χαρακτήρες "//" ενώ τα σχόλια πολλαπλών γραμμών ξεκινάνε με τους χαρακτήρες "/*" και τερματίζουν στους χαρακτήρες "*/"

Γραμμή 3η: Οι εντολές που ξεκινούν με "#" αποτελούν οδηγίες προς τον προεπεξεργαστή. Στο συγκεκριμένο παράδειγμα το πρόγραμμα ζητάει από τον προεπεξεργαστή να συμπεριλάβει την βιβλιοθήκη iostream η οποία περιέχει αντικείμενα και συναρτήσεις για την είσοδο και την έξοδο δεδομένων.

Γραμμή 5η: Χρήση του χώρου ονομάτων std. Ο χώρος ονομάτων std περιέχει πληθώρα συναρτήσεων, κλάσεων και αντικειμένων

Γραμμή 7η: Ορίζεται η κύρια συνάρτηση του προγράμματος δηλαδή το σημείο από το οποίο θα εκκινήσει η εκτέλεση του. Η συνάρτηση main έχει πάντα τύπο επιστροφής int

Γραμμή 9η: Εκτύπωση της συμβολοσειράς "Hello C++ World" μέσω του αντικειμένου cout το οποίο ορίζεται στο χώρο ονομάτων std της βιβλιοθήκης iostream. Ο χαρακτήρας διαφυγής "\n" δηλώνει την αλλαγή γραμμής

Γραμμή 10η: Τερματισμός προγράμματος επιστρέφοντας στο λειτουργικό σύστημα την τιμή 0. Η τιμή 0 ενημερώνει το λειτουργικό σύστημα ότι το πρόγραμμα τερματίστηκε κανονικά

3) Τύποι δεδομένων

Στη γλώσσα προγραμματισμού C++, υπάρχουν διάφοροι τύποι δεδομένων που χρησιμοποιούνται για να αποθηκεύσουν και να επεξεργαστούν δεδομένα. Ορισμένοι από τους βασικούς τύπους δεδομένων στη C++ περιλαμβάνουν:

char: Αυτός είναι ο τύπος δεδομένων για την αποθήκευση ενός χαρακτήρα. Για παράδειγμα: char letter = 'A';

bool: Αυτός είναι ο τύπος δεδομένων για τις λογικές τιμές true και false. Χρησιμοποιείται για την αναπαράσταση λογικών συνθηκών. Για παράδειγμα: bool is_true = true;

int: Αυτός είναι ο τύπος δεδομένων για τους ακέραιους αριθμούς. Χρησιμοποιείται για την αποθήκευση των θετικών και αρνητικών ακεραίων. Για παράδειγμα: int a = 10;

short (int): Ο τύπος short int, συνήθως απλά συντομογραφείται ως short, χρησιμοποιείται για την αποθήκευση ακεραίων αριθμών με μικρότερη ακρίβεια από τον τύπο int. Η ακρίβεια του short μπορεί να διαφέρει ανάλογα με τον μεταγλωττιστή, αλλά συνήθως έχει μέγεθος 16 bits(2 bytes), οπότε μπορεί να αποθηκεύσει ακέραιους αριθμούς από -32,768 έως 32,767. Για παράδειγμα: short small_integer = 100;

long (int): Ο τύπος long int, συνήθως συντομογραφείται ως long, χρησιμοποιείται για την αποθήκευση ακεραίων αριθμών με μεγαλύτερη ακρίβεια από τον τύπο int. Ο long έχει τουλάχιστον 32 bits(4 bytes) μεγέθος και μπορεί να αποθηκεύσει ακέραιους αριθμούς από περίπου -2,147,483,648 έως 2,147,483,647. Για παράδειγμα: long int big_number = 2147483648L;

float: Αυτός είναι ο τύπος δεδομένων για τους κινητούς δεκαδικούς αριθμούς με μικρότερη ακρίβεια. Για παράδειγμα: float pi = 3.14;

double: Αυτός είναι ο τύπος δεδομένων για τους κινητούς δεκαδικούς αριθμούς με υψηλότερη ακρίβεια από το float. Για παράδειγμα: double e = 2.71828;

long double: Ο τύπος long double χρησιμοποιείται για την αποθήκευση δεκαδικών αριθμών με μεγαλύτερη ακρίβεια από τον τύπο double. Η ακρίβεια του long double είναι εξαιρετικά υψηλή και μπορεί να χρησιμοποιηθεί για εφαρμογές που απαιτούν ακριβείς υπολογισμούς. Για παράδειγμα: long double pi = 3.14159265358979323846L;

Όσων αφορά τους τύπους long int και long double παρατειρούμε πως στο τέλος τους οι αριθμοί έχουμε ένα κεφαλαίο L, αυτό υπάρχει για να σηματοδοτεί στον μεταγλωττιστή πως είναι τέτοιου τύπου (long). Είναι σημαντικό να σημειωθεί ότι το ακριβές μέγεθος και ακρίβεια αυτών των τύπων μπορεί να διαφέρει ανάλογα με τον μεταγλωττιστή και το σύστημα. Καλό είναι να χρησιμοποιείτε αυτούς τους τύπους με προσοχή ανάλογα με τις ανάγκες του προγράμματός σας.

Επιπλέον, η C++ επιτρέπει τη δημιουργία προσαρμοσμένων τύπων δεδομένων με τη χρήση των δομών (structs) και των κλάσεων (classes) για να οργανώνετε τα δεδομένα σας σε πιο πολύπλοκες δομές. Αυτό σας επιτρέπει να δημιουργήσετε προσαρμοσμένες τύπους δεδομένων που αντιπροσωπεύουν περισσότερο σύνθετες έννοιες.

Τέλος παραπάνω οι τύποι δεδομένων απευθύνονται σε μεταβλητές. Στην C++ όμως υπάρχουν και σταθερές των ίδιων τύπων δεδομένων. Δηλώνονται όπως και οι μεταβλητές απλά με τη λέξη κλειδί const στο μπροστινό μέρος της δήλωσης. Για παράδειγμα:

const int x = 5; // δήλωση και αρχικοποίηση

Στην πάνω δήλωση λοιπόν το 'x' θα παραμείνει σταθερό με τιμή 5 κατά τη διάρκεια όλης της εκτέλεσης ενός προγράμματος. Αν ο προγραμματιστής προσπαθήσει να την αλλάξει μετά την αρχικοποίηση της θα υπάρξει πρόβλημα κατά τη μεταγλώττιση του προγράμματος.

Όνομα τύπουBytesΕύρος

Αξίζει να σημειώσουμε ότι αυτές οι τιμές μπορεί να ποικίλουν ανάλογα με το λειτουργικό σύστημα και την αρχιτεκτονική του υπολογιστή σας. Για ακριβείς τιμές σε μια συγκεκριμένη πλατφόρμα, μπορείτε να ανατρέξετε στην τεκμηρίωση της πλατφόρμας σας ή να χρησιμοποιήσετε τις σταθερές std::numeric_limits που παρέχονται από τη βιβλιοθήκη 'limits' στη C++.

4) Είσοδος-Έξοδος

Στη C++, η είσοδος (input) και η έξοδος (output) χειρίζονται κυρίως μέσω των αντικειμένων cin και cout, τα οποία προέρχονται από τη βιβλιοθήκη iostream. Ας δούμε πώς μπορείτε να χρησιμοποιήσετε αυτά τα αντικείμενα για είσοδο και έξοδο.

Έξοδος με το cout:

Το cout χρησιμοποιείται για την εκτύπωση δεδομένων στην οθόνη. Χρησιμοποιείται συνήθως με τον τελεστή <<.

Εδώ είναι ένα παράδειγμα:

                     
                        #include <iostream>

                        int main() {
                            std::cout << "Αυτό είναι ένα μήνυμα." << std::endl;
                            return 0;
                        }
Στο παραπάνω παράδειγμα, το κείμενο "Αυτό είναι ένα μήνυμα." εκτυπώνεται στην οθόνη.

Είσοδος με το cin:

Το cin χρησιμοποιείται για τη λήψη εισόδου από τον χρήστη. Χρησιμοποιείται με τον τελεστή >>.

Παράδειγμα:

                     
                        int main() {
                            int num;
                            std::cout << "Παρακαλώ εισάγετε έναν ακέραιο αριθμό: ";
                            std::cin >> num;
                            
                            std::cout << "Εισάγατε τον αριθμό: " << num << std::endl;
                            return 0;
                        }
Στο παραπάνω παράδειγμα, ο χρήστης καλείται να εισάγει έναν ακέραιο αριθμό, έστω το 3, ο οποίος αποθηκεύεται στη μεταβλητή num και στη συνέχεια εκτυπώνεται στην οθόνη.

Αυτό είναι ένα απλό παράδειγμα της χρήσης των cin και cout.

Σε πιο πολύπλοκα προγράμματα, μπορείτε να χρησιμοποιήσετε αυτά τα αντικείμενα για να διαχειριστείτε πολλαπλές εισόδους και εξόδους.

5) Τελεστές

Στη C++, υπάρχουν διάφοροι τύποι τελεστών που χρησιμοποιούνται για διάφορους σκοπούς. Ας δούμε τις τρεις κύριες κατηγορίες τελεστών. Αρχικά έχουμε τους αριθμητικούς τελεστές οι οποίοι χρησιμοποιούνται για αριθμητικές πράξεις πάνω σε ακέραιους και δεκαδικούς αριθμούς. Ορισμένοι από τους αριθμητικούς τελεστές περιλαμβάνονται στο παρακάτω πινακάκι.

5.1) Αριθμητικοί τελεστές

ΤελεστήςΠεριγραφήΠαράδειγμα
+Πρόσθεσηx + y
-Αφαίρεσηx - y
*Πολλαπλασιασμόςx * y
/Διαίρεσηx / y
++(προθεματικό)Αύξηση κατά ένα και επιστροφή της νέας τιμής++x
++(μεταθεματικό)Αύξηση κατά ένα και επιστροφή της παλιάς τιμήςx++
--(προθεματικό)Μείωση κατά ένα και επιστροφή της νέας τιμής--x
--(μεταθεματικό)Μείωση κατά ένα και επιστροφή της παλιάς τιμήςx--

Έστω παράδειγμα του τελεστή της πρόσθεσης:

                       
                        int a = 5;									
                        int b = 3;
                        int result = a + b; // Πρόσθεση
Αν εμφανίσουμε τώρα την τιμή της μεταβλητής result τότε θα έχει το αποτέλεσμα της πρόσθεσης των a,b δηλαδή 3,5: 3+5 = 8. Άρα, το result θα έχει την τιμή 8. Αντίστοιχα χρησιμοποιούμε και τους υπόλοιπους αριθμητικούς τελεστές.

5.2) Σχεσιακοί τελεστές

Οι σχεσιακοί τελεστές χρησιμοποιούνται για να συγκρίνουν τιμές και επιστρέφουν αποτελέσματα true ή false (αληθές ή ψευδές , όπου 0 είναι false και ό,τι δεν είναι θεωρείτε true.Συνήθως το 1 είναι το true.). Οι σχεσιακοί τελεστές φαίνονται στον ακόλουθο πίνακα:

ΤελεστήςΠεριγραφήΠαράδειγμα
==Ισούται μεx == y
!=Δεν ισούται μεx != y
>Μεγαλύτερο απόx > y
<Μικρότερο απόx < y
>=Μεγαλύτερο από ή ίσοx >= y
<=Μικρότερο από ή ίσοx <= y

Έστω παράδειγμα του τελεστή > (μεγαλύτερο από):

                     
                        int a = 5;
                        int b = 3;
                        bool r = a > b; // Είναι το a μεγαλύτερο από το b;
Εδώ το a είναι το 5 και b το 3 οπότε ναι η σύγκριση είναι αληθής και άρα η τιμή της bool (boolean) μεταβλητής r θα έχει ως τιμή 'true'.

5.3) Λογικοί τελεστές

Οι λογικοί τελεστές χρησιμοποιούνται για λογικές πράξεις, όπως ο συνδυασμός δυο ή περισσότερων συνθηκών. Οι λογικούς τελεστές περιλαμβάνονται στον παρακάτω πίνακα:

ΤελεστήςΠεριγραφήΠαράδειγμα
&&Λογικό καιx < 5 && y > 10
||Λογικό ήx < 5 || y > 10
!Άρνηση!(x < 5 && y > 10)

Έστω παράδειγμα χρήσης αυτών των τελεστών σε απλό κώδικα με bool μεταβλητές: 

                    
                    bool t = true;
                    bool f = false;
                    bool res = t && f; // Λογική σύζευξη (AND)

Εδώ αν εμφανίσουμε την τιμή του res θα δούμε πως έχει την τιμή false εφόσον true AND false = false. Αντίστοιχα μπορούμε να τους χρησιμοπιήσουμε σε συνδυασμό με τους σχεσιακούς τελεστές σε εκφράσεις με αριθμούς (ή χαρακτήρες και αλφαρηθμητικά που θα δούμε στη συνέχεια σε επόμενο κεφάλαιο.) όπως φαίνονται στα παραδείγματα των πινάκων στην τελευταία στήλη. Βέβαια η αλήθεια είναι πως αυτό το παράδειγμα δεν είναι πρακτικό εφόσον στην πραγματικότητα δεν χρησιμοποιούμε αυτούς τους τελεστές με αυτόν τον τρόπο αλλά σε <<συνθήκες>> οι οποίες είναι μέρος των δομών ελέγχου και επανάληψης που θα δούμε σε επόμενο κεφάλαιο.

Οι τελεστές αυτοί είναι βασικοί για τον προγραμματισμό στη C++ και χρησιμοποιούνται ευρέως για να εκτελέσουν διάφορους τύπους πράξεων και συγκρίσεις στον κώδικα.
