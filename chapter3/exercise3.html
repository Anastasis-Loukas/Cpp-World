1. Έστω το παρακάτω πρόγραμμα

#include <iostream>

using namespace std;

int main()
{
  int a, b, temp, *c, *d;
  c = &a;
  d = &b;
  int &e = b;
  int &f = *c;

  a = 5;
  b = 10;

  temp = *c;
  *c = *d;
  *d = temp;

  d = c
  *d = 6;

  e--;
  f++;

  return 0;
}

Συμπληρώστε τις τελικές τιμές αφού εκτελεστεί το πρόγραμμα

Έστω ότι οι διευθύνσεις της στοίβας ξεκινάνε από τον αριθμό 1000, κάθε ακέραιος έχει μέγεθος 4 byte και το λειτουργικό σύστημα στο οποίο εκτελείται το πρόγραμμα είναι 64 bit. Ποιές θα είναι οι τιμές των διευθύνσεων μετά την εκτέλεση του προγράμματος
